# About the Project

- This is a static website for a handcrafted products store.
The platform is designed as a product showcase, where users can browse items and collections.

## Tech Stack
- Next.Js
- Shadcn 

## Requirements
- Node 20+

## Running Project

Install dependencies:

npm install
or
yarn
or
pnpm install
or
bun install

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
